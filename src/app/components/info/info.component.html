This small app provides conversation starters.

This app is an Angular UI. For styling, It uses Bootstrap 4 (with SCSS for custom styling) along with Font Awesome 4 icons. I made the Favicon with Metro Studio.

This app relies on a simple Express API hosted at https://aoti-basic-express-app.herokuapp.com/.

The frontend (Angular) and backend (Express) are both hosted on Heroku.

This repo for this project can be found on github at https://github.com/adam-on-the-internet/chit-chat.
